# FormInputWrapper
`<FormInputWrapper>` is used to wrap a form's inputs to (usually) display error messages.

It is composed of the first example in `<FormGroup>` doc, with:

`<FormGroup>` to contains all the elements

`<FormGroupLabel>` to display a title above the input

`<FormGroupTextHelp>` to display a message below the input
